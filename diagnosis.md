# Диагностика проблем с document.write()

## Проверки на стороне клиента:

### 1. Режим инкогнито
Откройте сайт в режиме инкогнито:
- Chrome: Ctrl+Shift+N
- Firefox: Ctrl+Shift+P

### 2. Отключить расширения
Временно отключите:
- AdBlock, uBlock
- VPN клиенты
- Защитники браузера
- Встроенные скрипты

### 3. Очистка кэша
```
Ctrl+F5 (жесткое обновление)
или
Ctrl+Shift+Delete (очистка кэша)
```

### 4. Проверка консоли браузера
F12 → Console → найти:
- Ошибки document.write()
- Скрипты с oartatic.net
- Медленные загрузки

### 5. Сетевая диагностика
В Developer Tools → Network:
- Проверить загружаемые ресурсы
- Найти источники оartatic.net

## Диагностические команды:
```bash
# Проверка сайта через curl
curl -s http://localhost/ | grep -i "document.write\|oartatic"

# Проверка заголовков
curl -I http://localhost/

# Анализ исходного кода
curl -s http://localhost/ > page.html
```

## Временное решение:
Если проблема критична, можно отключить все внешние скрипты:
1. Локально разместить Bootstrap
2. Убрать CDN ссылки
3. Использовать только локальные файлы

- ✅ Современные методы загрузки скриптов
- ✅ Используется async/defer для внешних ресурсов
- ✅ Все сервисы работают стабильно
